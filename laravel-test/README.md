## Set up the Collationhub Coding Round Task
### Pre- reqirements 
1. Xampp must be available (for php and mysql) if not xampp php and my sql must be available.
2. Node must be installed 
3. Composer must be installed

### Process to run
1. Clone from git , than cd laravel-test
2. Set up .env by coping the  .env.example
3. Run Composer install 
4. Create a table in phpmyadmin name="Laravel-test"
5. Run php artisan migrate
6. Run php artisan serve

### How to check the APIs
1. Go to post import the environment and collection (JSON files provided on git)
2. Try to run the Apis provided in Collection
